[![GitHub last commit](https://img.shields.io/github/last-commit/assistant-ui/tool-ui.svg?style=flat-square&maxAge=5184000)]()
[![GitHub tag](https://img.shields.io/github/tag/assistant-ui/tool-ui.svg?style=flat-square&maxAge=5184000)]()

[![GitHub issues](https://img.shields.io/github/issues/assistant-ui/tool-ui.svg?style=flat-square&maxAge=5184000)]()

Copy/paste React components for rendering tool calls in AI chat interfaces.
When a model calls a tool, most apps dump raw JSON into the conversation. These components turn tool payloads into
interactive UI like approvals, forms, tables, charts, and media cards so users can understand and act without leaving
the chat.
